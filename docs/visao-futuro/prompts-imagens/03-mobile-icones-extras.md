## Dicas de Uso no NanoBanana

1. **Prefixo Consistente**: Adicione sempre "Desktop application UI, professional business software, clean design, Windows 11 style" ao início
2. **Resolução**: Especifique "1920x1080" ou "4K resolution" para telas completas
3. **Estilo**: Use termos como "modern flat design", "minimalist UI", "professional corporate"
4. **Cores**: Mencione a paleta quando relevante: "blue accent #2563EB", "green success #22C55E"
5. **Contexto**: Inclua "Brazilian Portuguese text" para labels em português
6. **Qualidade**: Adicione "high quality render", "crisp UI elements", "sharp text"
